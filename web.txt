Part 6: Explain How the Web Works
How does the internet work? What happens when you click on a link?

The internet works when we connect to the internet through browsers and examples of browsers are Chrome, Safari, Firefox, etc. We access these different browsers on our computers (servers) where they are installed. Our computers send web requests and receive web responses. They also read CSS and HTML and JavaScript. HTML stands for Hypertext Markup Language and is the standard markup language for creating Web pages. It describes the structure of a Web page and tells the browserm how to display the content. Designers create a wide range of web pages that can be read by different kinds of browsers. We code in JavaScript to work with webpages through HTML and CSS (Cascading Style Sheets). Different complicated mathematical operations are done with large amounts of data around the world. Computers/ servers are all connected and there are countless different components that work together to display webpages on our browsers.
